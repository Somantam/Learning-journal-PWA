Week 4: API Integrations

Demonstration of implementing Storage, Browser, and Third-Party APIs to add persistence, notifications, and external content to the PWA.

  • Storage API (Local Storage): Used for theme preference persistence and saving recent journal entries (Journal page).
  • Browser API (Notifications API): Used on the Journal page to prompt for and send a confirmation notification upon saving data.
  • Third-Party API (ZenQuotes API): Used on the Homepage to fetch and display a random motivational quote.
