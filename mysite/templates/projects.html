Mini Project: Focus Timer

A personal productivity tool integrated directly into the Learning Journal PWA. This Pomodoro-style timer helps users manage study sessions with persistent tracking and notifications.

  • State Persistence: Solved the challenge of resetting timers by saving the "Target Timestamp" to Local Storage, allowing the countdown to continue accurately even if the user navigates away or refreshes.
  • JavaScript Logic: Utilized setInterval and Date objects to calculate remaining time in real-time.
  • Notifications: Integrated the Browser Notifications API to alert users when a session is complete, even if they are viewing a different tab.
  • UI/UX: Designed a clean, distraction-free interface with custom CSS to ensure visibility and accessibility.

Technologies: JavaScript (Intervals, Date), Local Storage API, Notifications API, CSS, Flask Routing

Lab 7: Progressive Web App (PWA)

Transformed the Learning Journal into a fully installable Progressive Web App with offline capabilities and network independence.

  • Service Worker: Implemented sw.js to cache shell assets (HTML, CSS, JS) for instant loading.
  • Offline Support: Configured the Cache API to serve content even when the user has no internet connection.
  • Installability: Added a manifest.json file with custom icons (`192px` & `512px`), allowing the app to be installed on mobile and desktop.
  • Extra Feature: Built a real-time Offline Notification System that alerts users via a sticky banner when connectivity is lost.
  • Backend Logic: Created a custom Flask route to serve the Service Worker from the root scope (`/sw.js`) to ensure full app control.

Technologies: Service Workers, Cache API, Web App Manifest, Flask, JavaScript Event Listeners.
